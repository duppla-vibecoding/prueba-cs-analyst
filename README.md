# Prueba CS Analyst · Duppla

Prueba de 30 min / 10 preguntas para candidatos al rol de CS Analyst.

- **Form público**: https://duppla-vibecoding.github.io/prueba-cs-analyst/
- **Respuestas**: Google Sheet (privado) — se escriben vía webhook n8n
- **Arquitectura**: HTML vanilla + GH Pages + webhook n8n (text/plain, sin CORS preflight) + Google Sheets

Para ejecutar localmente: abrir `index.html` en el browser.
